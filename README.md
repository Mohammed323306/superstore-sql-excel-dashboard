# 📊 Superstore Sales Analysis Dashboard
## SQL (SQLite) + Excel Project

### 📌 Project Overview

This project analyzes Superstore sales data using SQLite (DB Browser) for data transformation and Microsoft Excel 365 for dashboard visualization.

The goal was to evaluate sales performance across regions, categories, products, shipping modes, and time to generate business insights.

### 🛠 Tools Used

SQLite (DB Browser)

SQL (aggregations, views, KPI analysis)

Microsoft Excel 365 (PivotTables, Slicers, Dashboard)

## 📊 Key Insights

### 🌍 Regional Performance

West generated the highest sales: 710,220

Followed by East (669,519), Central (492,647), and South (389,151)

### 🏷 Category Performance

Technology is the top-performing category with 827,456

Furniture: 728,659

Office Supplies: 705,422

### 📅 Sales Trends

Strong growth from 2016 to 2018

2018 Q4 recorded the highest quarterly sales (278,416)

Q4 consistently shows peak seasonal performance

### 📦 Shipping Analysis

Standard Shipping has the highest average delivery time (5 days)

### 🏆 Top Selling Products

Canon imageCLASS 2200 Advanced Copier – 61,600

Fellowes PB500 Binding Machine – 27,453

Cisco TelePresence System EX90 – 22,638

HON 5400 Series Task Chairs – 21,871

HP Designjet T520 Printer – 18,375

## 📈 Dashboard Features

KPI Cards (Total Sales, Orders, Customers, Avg Ship Days)

Monthly & Quarterly Sales Trend

Sales by Region

Category & Sub-category Analysis

Top 10 Products

Interactive Slicers (Region, Segment, Category, Year)

All visuals dynamically update based on slicer selections.

## 🔄 How to Reproduce

Import the dataset into SQLite (DB Browser)

Run SQL scripts in the /sql folder

Export cleaned dataset

Open Superstore_Dashboard.xlsx

Refresh PivotTables if needed
